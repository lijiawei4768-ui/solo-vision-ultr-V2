重新定义：Fretboard Literacy Trainer
在不同音乐约束下，快速、稳定、无形状依赖地定位指板上的音高信息。不只是找字母 — 从 Note 到 Degree 到 Chord Tone 到 Context。

A

B

D
Pressure — 反应训练
限时、连击、节奏触发、错误惩罚。把"认识"变成"反应"。

重新定义：Scale Systems Trainer
训练"你如何在不同系统里组织、连接、理解并应用音阶"的能力。不是背几个盒子 — 是 CAGED / 3NPS / Horizontal / Phrase Cells 等系统思维的真正训练。
模式树 · 第一阶段锁 5 个
A
Pattern — 单系统单形态
选系统、选音阶、选位置，练最基本的落点。当前代码 seqMode ascending/descending/random 是这里最原始的雏形。
B
System — 系统比较与切换
同一音阶在 CAGED / 3NPS / Horizontal / Hybrid 中的不同组织方式。这是你新增 3NPS/CAGED/JTC 系统的核心入口 — 必须进 L1 主位。
C
Sequence — 运行逻辑与 pattern cell
3-note、4-note、skip、thirds、1235、1234 等。当前 seqMode 只是雏形，这里要做真正的 sequence logic。
D
Connection — 位置连接与跨把位
相邻 shape 连接、区域连接、跨把位移动。CAGED 相邻 shape / 3NPS 跨弦跳进等。
E
Target Tone — 功能音定位
在 scale 内找 3rd / 7th / guide tones / tensions / avoid notes。和 Chords 接轨的关键桥梁模式。
Mode F (Chord-Scale) 暂缓，等前 5 个成熟后再加。
系统层定义（你要新增的核心）
System A
CAGED
五个位置 · 相邻位置连接 · 和弦锚点 · 位置认知 · chord-scale 关系
System B
3NPS (3 Notes Per String)
三音每弦 · 横向对称 · 流动性 · 快速 sequence · 跨把位移动
System C
Horizontal
单弦/双弦/横向路线 · 解决"只会在 box 里想"的问题 · 培养线条感而非格子记忆
System D
Phrase Cells / Hybrid
interval cells · sequence fragments · pattern chunks · 吸收 JTC/Tom Quayle 式思维，但产品内用中性系统名
产品内部不要直接标注"JTC 模式"，用系统中性名称。Phrase Cells / Hybrid 可以第二阶段再加，先把 CAGED + 3NPS + Horizontal 做扎实。
